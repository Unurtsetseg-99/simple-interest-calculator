# Simple Interest Calculator

Энэ төсөл нь **энгийн хүү** тооцоолох calculator юм.

## Формула
**Simple Interest (SI)** = (Principal × Rate × Time) / 100

## Хэрэглэх заавар
1. Principal (үндсэн төлбөр) оруул
2. Interest Rate (%) оруул
3. Time (жил) оруул
4. Calculate дар

## Жишээ
- Principal = 10000
- Rate = 5%
- Time = 2 жил
- **Simple Interest = 1000**
